## Datasets

This folder contains the raw datasets used to build the Power BI and Tableau dashboards in this repository.  
The datasets are organized by visualization tool to maintain clarity and reproducibility.

These files represent structured sales data commonly used for business analytics and reporting use cases.

## Power BI Datasets  
**Folder:** `Power-BI/`

The following datasets are used for building the **Sales Analytics Dashboard** in Power BI:

- `SuperStore Sales DataSet.xlsx`  
- `SuperStore_Sales_Dataset.csv`  

### Description
These datasets contain transactional sales information including:
- Order details  
- Sales and profit values  
- Product and category information  
- Customer and regional attributes  

They are suitable for creating time-based trends, category analysis, and regional performance visuals.

## Tableau Datasets  
**Folder:** `Tableau/`

The following datasets are used for building the **Customer Sales Dashboard** in Tableau:

- `Customers.csv`  
- `Orders.csv`  
- `Products.csv`  
- `Location.csv`  

### Description
These datasets are modeled using a relational structure:
- Customers → Orders → Products  
- Location data for geographic analysis  

This structure enables customer-level analysis, segmentation, and contribution insights.

## Notes
- These datasets are used for learning and portfolio projects  
- Minor cleaning and transformation are performed within Power BI and Tableau  
- No sensitive or proprietary data is included
