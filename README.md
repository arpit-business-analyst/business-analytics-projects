# Business Analytics Projects

This repository contains my learning and project work in SQL, Excel, and Business Analysis.

## SQL Mini Project: World Data Analysis

Objective:  
To analyze global country data and answer common business-style analytical questions using SQL.

Dataset:  
World table (country-level data including population, continent, GDP, and area)

Key Questions Answered:
- Countries with population greater than 100 million
- Total population by continent
- Top 5 most populated countries
- Countries with population above world average
- Continents having more than 10 countries

Skills Demonstrated:
- SQL SELECT queries
- Filtering and sorting
- GROUP BY and HAVING
- Subqueries
- Analytical thinking
